Questão 1: Explique o que é CGI

Um CGI (Commun Gateway Interface ) é uma tecnologia que permite gerar páginas dinâmicas nas web, deste modo, o navegador passar os parâmetros para um programa localizado em um servidor web, para que o mesmo processe esses parâmetros e retorne uma resposta.

O CGI funciona da seguinte maneira: O cliente (browser) solicita uma URL ao servidor; a URL solicitada é referente a um CGI, então o servidor executa o CGI; O CGI então trabalha podendo interagir com outras aplicações, recupera os dados e os retorna para o servidor; o servidor envia os dados para o cliente, que formata o resultado e apresenta ao usuário.
