Questão 5: Explique o que é CSS3

O CSS (Cascading Style Sheets ) é uma ferramenta que permite adicionar estilos como cores, fontes, espaçamentos a um documento web, tornando-o mais bonito visualmente. O CSS3 é a última versão do CSS e que adiciona ao leque de opções efeitos de transição, imagem, dentre outros.

Existem algumas bibliotecas de CSS3 que já possuem estilos e componentes pré-definidos, assim tornando mais rápido o desenvolvimento de uma aplicação, alguns exemplos dessas bibliotecas são: MaterializeCSS, Bootstrap e Material Design Lite.
