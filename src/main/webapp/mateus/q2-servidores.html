Questão 2: Explique o que é Servidor Web, Container Web (Java) e Servidor de Aplicação

O servidor web é responsável por responder todas as solicitações feitas para um endereço na internet. O mais popular e utilizado no mundo atualmente é o servidor Apache (software livre).

O container web é o componente de um servidor web que interage com o servlets Java. Esse container é responsável por gerenciar o ciclo de vida de servlets, mapear uma URL para um servlet particular e garantir que o requisitante da URL possua os direitos corretos. O container web implementa a arquitetura do Java EE.

O servidor de aplicação é um servidor que disponibiliza um ambiente para instalação e execução de determinadas aplicações, assim dispensando a necessidade de instalação nos computadores dos usuários. Os servidores de aplicação também são conhecidos como middleware.
